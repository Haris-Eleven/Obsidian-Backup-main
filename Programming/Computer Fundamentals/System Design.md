# Web App Architecture 
## Serverless
- Uses cloud-managed services (like AWS Lambda, Azure Functions) for backend logic
## Microservices
- Application is broken into small, independent services that communicate over HTTP/gRPC
## Monolithic
- Single, unified codebase that handles the UI, business logic, and data access layer.
